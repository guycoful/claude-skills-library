1 מה צריך לפני שמתחילים?

חשבון Google AI Studio

הסקיל משתמש ב-Google Gemini API. כדי להשתמש בו צריך מפתח API מגוגל.

💰 כמה זה עולה?

בתשלום לפי שימוש. בעת יצירת מפתח API ב-AI Studio, צריך לחבר את הפרויקט לחשבון Billing עם כרטיס אשראי. המדריך למטה מסביר בדיוק איך לעשות את זה.

2 קבלת מפתח API מגוגל

  1. היכנסו ל-Google AI Studio

    לחצו על הכפתור:

    🔑 היכנסו ל-Google AI Studio
  2. התחברו עם חשבון Google

    השתמשו בחשבון הגוגל הרגיל שלכם (Gmail).

  3. צרו מפתח API חדש

    לחצו על Create API Key (צור מפתח API).

  4. בחרו פרויקט

    אם יש לכם כבר פרויקט - בחרו אותו. אם לא - לחצו על Create API key in new project.

  5. הגדרת Billing (חיוב)

    ה-API של Gemini דורש חיבור לחשבון בתשלום. אם יצרתם פרויקט חדש, תראו בטבלה קישור בכחול: Set up billing.

    1. לחצו על Set up billing - ייפתח Google Cloud Console
    2. לחצו על Create account ליצירת חשבון חיוב חדש
    3. הזינו פרטי אשראי ואשרו
    4. החשבון יקושר אוטומטית לפרויקט שיצרתם
    💡

    אם יש לכם כבר חשבון Billing ב-Google Cloud, תוכלו פשוט לבחור אותו מהרשימה במקום ליצור חדש.

  6. העתיקו את המפתח

    יופיע מפתח ארוך. לחצו על כפתור ההעתקה ושמרו אותו.

    המפתח נראה כמו: AIzaSy...

🔒

שמרו על המפתח! אל תשתפו אותו ואל תעלו אותו לאינטרנט.

⚠️ לפני שממשיכים - הגדרת פרטיות

בשלב הבא תזינו מפתח API. זה כמו סיסמה, ולכן חשוב לוודא שהגדרות הפרטיות מוגדרות נכון בקלוד קוד.

🔐
