ה-Pipeline המלא

הסקיל עובד ב-6 שלבים, כל אחד משתמש בסקיל ייעודי:

1. Content - presentation-architect

יוצר תוכנית מצגת מפורטת: כותרות, תוכן, כיוון ויזואלי לכל שקף

2. HTML - בניית מצגת

HTML יחיד ב-1920x1080, ניווט מקלדת, תמיכה ב-RTL עברית

3. Images - image-generation

פוסטר AI לכל שקף (Gemini/FLUX), כל התמונות במקביל

4. Videos - image-generation (video)

Grok Imagine Video - ממיר כל תמונה לוידאו לופ. מצלמה סטטית בלבד!

5. Deploy - gh-pages-deploy

מפרסם הכל ל-GitHub Pages עם URL חי

6. Share - whatsapp (אופציונלי)

  1. היכנסו לקונסול xAI 🌐 xAI Console
  2. צרו מפתח API

    מקבלים $25 קרדיט חינם להתחלה!

GitHub CLI
