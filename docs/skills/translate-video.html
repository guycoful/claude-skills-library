Translate Video

תרגמו סרטונים לכל שפה עם כתוביות איכותיות ברמת דובר שפת אם

⏱️ זמן התקנה: 5 דקות (אם יש כבר transcribe)

מה הסקיל הזה עושה?

צינור שלם מקצה לקצה: מקבל סרטון, מתמלל, מתרגם, משפר את התרגום לרמת דובר שפת אם, מתקן בעיות כיוון כתיבה RTL, ושורף את הכתוביות על הסרטון. הכל אוטומטי.

🎯 הבעיה שנפתרה: כיוון כתיבה RTL

אחד האתגרים הכי מעיקים בהטמעת כתוביות בעברית הוא כיוון הכתיבה. כשיש מילים באנגלית (כמו שמות מוצרים), נקודות ופסיקים - הכל מתבלבל.

הפתרון

הוספת Unicode marks אוטומטית לכל שורת טקסט בקובץ SRT:

  • U+202B (RLE) - מכריח כיוון RTL לפסקה
  • U+202C (PDF) - סוגר את ה-embedding
✅

התוצאה: כתוביות בעברית שנראות מושלם - גם עם מילים באנגלית, מספרים, ופיסוק מעורב.

נדרש מפתח API של ElevenLabs (לתמלול). מוגדר בסקיל transcribe.

  3. וודאו שסקיל embed-subtitles מותקן

    ffmpeg -version

🔧 פתרון בעיות נפוצות

הקובץ גדול מדי לתמלול

  • הסקיל מחלץ אוטומטית את האודיו לקובץ MP3 קטן יותר
  • אם עדיין נכשל, נסו לחלץ ידנית: ffmpeg -i video.mp4 -vn -ab 64k audio.mp3

הכתוביות בכיוון הפוך

  • הסקיל מטפל בזה אוטומטית עם Unicode RTL marks
  • אם עדיין יש בעיה, וודאו שהקובץ נשמר ב-UTF-8

התרגום לא טבעי

  • הסקיל כולל שלב שיפור סמנטי שמבטיח טבעיות
  • אם צריך שינויים, ערכו את קובץ ה-SRT המתורגם והריצו רק את שלב ההטמעה
