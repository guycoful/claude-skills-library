1 יצירת מפתח API של Gemini (בתשלום)

מינואר 2026 אפשר להגדיר חיוב ישירות מתוך Google AI Studio - בלי צורך ב-Google Cloud Console!

  1. היכנסו ל-Google AI Studio

    גלשו לאתר:

    🌐 פתחו את AI Studio
  2. גשו לדף ה-API Keys

    בתפריט הצד לחצו על API Keys או גלשו ישירות ל: aistudio.google.com/apikey

  3. צרו מפתח API

    לחצו Create API Key. המפתח נראה כמו: AIzaSy...

  4. הפעילו חיוב (Billing)

    בעמודת Quota tier ליד הפרויקט, לחצו על Set up Billing או Upgrade.

    לחלופין: Dashboard → Usage and Billing → Billing tab.

  5. הזינו פרטי תשלום

    בחרו מדינה, הסכימו לתנאים, והוסיפו כרטיס אשראי עם כפתור ה-"+".

    לחצו Finish account setup.

  6. אישור תשלום מקדים (אם נדרש)

    גוגל עשויים לדרוש תשלום מקדים חד-פעמי ($5-$50) - זה לא תשלום אלא קרדיט שנכנס לחשבון שלכם לשימוש עתידי.

💡

מה מקבלים? שדרוג מיידי ל-Tier 1 (150-300 בקשות לדקה), פרטיות מלאה (גוגל לא משתמשים בנתונים שלכם לאימון), וגישה למודלים בתשלום.

2 מפתח fal.ai (אופציונלי - למצב זול)

אם אתם רוצים גם את המצב הזול (~$0.003 לתמונה), תצטרכו מפתח fal.ai:

  1. היכנסו ל-fal.ai 🌐 פתחו את fal.ai
  2. צרו חשבון והיכנסו

    הירשמו עם GitHub או Google.

  3. צרו מפתח API

    גלשו ל-Dashboard → Keys → צרו מפתח חדש.

3 מפתח xAI Grok (אופציונלי - לוידאו)

ליצירת וידאו באיכות גבוהה עם Grok Imagine Video:

  1. היכנסו ל-xAI Console 🌐 פתחו את xAI Console
  2. צרו מפתח API

    גשו ל-API Keys → צרו מפתח חדש.

⚠️ לפני שממשיכים - הגדרת פרטיות

בשלב הבא תזינו מפתחות API. אלה כמו סיסמאות, ולכן חשוב לוודא שהגדרות הפרטיות מוגדרות נכון בקלוד קוד.

🔒

אם עדיין לא עשיתם את זה, עברו קודם למדריך הגדרות פרטיות וודאו שכיביתם את האפשרות לאנתרופיק להתאמן על השיחות שלכם.

4 הגדרת הסקיל במחשב

הכי קל - לבקש מקלוד קוד לעשות את זה בשבילכם!

🪄 הדרך הקלה - בקשו מקלוד

פתחו את קלוד קוד ואמרו לו:

צור קובץ .env בתיקייה ~/.claude/skills/image-generation/scripts עם התוכן הבא: GEMINI_API_KEY=המפתח_שהעתקתם_מ_AI_Studio FAL_KEY=המפתח_מ_fal_ai XAI_API_KEY=המפתח_מ_xAI

קלוד ייצור את הקובץ במקום הנכון!

* רק GEMINI_API_KEY נדרש. השאר אופציונלי לפי הספקים שתרצו להשתמש בהם.

התקנת ספריות

אחרי שהקובץ נוצר, בקשו מקלוד:

הרץ npm install בתיקייה ~/.claude/skills/image-generation/scripts

5 בדיקה שהכל עובד

הפעילו מחדש את קלוד קוד ונסו:

# בקשו מקלוד: צור תמונה של עיר עתידנית בשקיעה

קלוד ייצור את התמונה ויפתח אותה. אם ראיתם תמונה - הכל עובד! 🎉

דוגמאות לשימוש

תמונות

# תמונה באיכות גבוהה (Gemini) "צור תמונה של הרים בשלג עם אור זהב" # תמונה זולה ומהירה (fal.ai) "צור תמונה זולה של חתול על ספה" # עם יחס גובה-רוחב "צור תמונה אנכית לסטורי של קפה בוקר" # עריכת תמונה קיימת "קח את התמונה הזו והפוך אותה לציור שמן" # תמונה עם טקסט עברי "צור פוסטר עם הכיתוב: חג שמח"

וידאו

# וידאו באיכות גבוהה (Grok) "צור וידאו של גלים מתנפצים על סלעים" # וידאו זול ומהיר (LTX-2) "צור וידאו זול של עננים נעים" # אנימציה של תמונה "קח את התמונה הזו ותעשה ממנה וידאו עם תנועה עדינה"

⚡ מצב WOW

רוצים אפקטים מטורפים? אמרו לקלוד "wow mode" והוא יוסיף אפקטים קולנועיים: חלקיקים, פיצוצי אנרגיה, lens flares, ברקים ועוד.

"צור תמונה wow mode של גיבור על שעומד על גג בניין"

🔧 פתרון בעיות נפוצות

שגיאת "GEMINI_API_KEY not found"

  • וודאו שיצרתם קובץ .env בתיקיית scripts
  • בדקו שהמפתח מתחיל ב-AIzaSy

שגיאת Quota / Rate Limit

  • וודאו שהפעלתם חיוב (Billing) ב-AI Studio
  • בדקו את ה-Tier שלכם בדף ה-API Keys

טקסט עברי יוצא הפוך

  • קלוד מוסיף אוטומטית הנחיית RTL - אם זה לא עובד, בקשו ממנו להדגיש RTL בפרומפט

רוצים להשתמש רק ב-Gemini?

  • מספיק רק GEMINI_API_KEY בקובץ .env
  • המפתחות האחרים נדרשים רק אם תשתמשו ב---cheap (fal.ai) או וידאו (xAI)
